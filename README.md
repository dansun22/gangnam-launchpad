# Gangnam Launchpad

8 audio samples are included in `sound_source/`.

## How to run

Option 1: open `index.html` directly in a browser.

Option 2, recommended: run `start-server.bat`, then open:

```text
http://127.0.0.1:8090/
```

The local server option is more reliable for audio playback because some browsers restrict local file audio access.

## Controls

- Click pads or press `A S D F J K L ;` to play samples.
- Click cells in `4 Bar Loop Tracks` to place samples in the loop.
- `Play Loop` starts the 4-bar sequencer.
- `Stop` stops the loop.
- `Clear` removes all placed steps.
- `Space` toggles loop playback.
- `Esc` stops playback.
